# Socketio chat
Examples about simple chats using socket.io

## Build

Simple chat using `Express`, `Socket.io`, `Jquery`, URI: `localhost:3000`.

```bash
$ cd basic-chat
$ npm i
$ npm start
```


Chat with a better structure `Express`, `Socket.io`, `Materialize`, URI: `localhost:3000`.


```bash
$ cd better-chat
$ npm i
$ npm start
```
